Multidimensional array

A multidimensional array is an array that contains another array

EXAMPLE:
const data = [[1, 2, 3], [1, 3, 4], [4, 5, 6]];

Create a Multidimensional Array
let studentsData = [['Jack', 24], ['Sara', 23], ['Peter', 24]];
let student1 = ['Jack', 24];
let student2 = ['Sara', 23];
let student3 = ['Peter', 24];

// multidimensional array
let studentsData = [student1, student2, student3];

EXAMPLE:
let x = [
['Jack', 24],
['Sara', 23],
['Peter', 24]
];

// access the first item
console.log(x[0]);

Output:
["Jack", 24]
// access the first item of the first inner array
console.log(x[0][0]);

Output:
Jack
// access the second item of the third inner array
console.log(x[2][1]);

Output:
24

Add an Element to a Multidimensional Array
You can use the Array's push() method or an indexing notation to add elements to a multidimensional array.

EXAMPLE:
let studentsData = [['Jack', 24], ['Sara', 23],];
studentsData.push(['Peter', 24]);
console.log(studentsData);

Output:
[["Jack", 24], ["Sara", 23], ["Peter", 24]

// using index notation let studentsData = [['Jack', 24], ['Sara', 23],];
studentsData[1][2] = 'hello';
console.log(studentsData);

Output:
[["Jack", 24], ["Sara", 23, "hello"]]

// using push()
let studentsData = [['Jack', 24], ['Sara', 23],];
studentsData[1].push('hello');
console.log(studentsData);

Output:
[["Jack", 24], ["Sara", 23, "hello"]]

You can also use the Array's splice() method to add an element at a specified index.
For example:
let studentsData = [['Jack', 24], ['Sara', 23],];

// adding element at 1 index
studentsData.splice(1, 0, ['Peter', 24]);
console.log(studentsData);

Output:
[["Jack", 24], ["Peter", 24], ["Sara", 23]]

Remove an Element from a Multidimensional Array

You can use the Array's pop() method to remove the element from a multidimensional array.
For example:

// remove the array element from outer array
let studentsData = [['Jack', 24], ['Sara', 23],];
studentsData.pop();
console.log(studentsData);

Output:
[["Jack", 24]]

// remove the element from the inner array
let studentsData = [['Jack', 24], ['Sara', 23]];
studentsData[1].pop();
console.log(studentsData);

Output:
[["Jack", 24], ["Sara"]]

You can also use the splice() method to remove an element at a specified index.
For example:
let studentsData = [['Jack', 24], ['Sara', 23]];

// removing 1 index array item
studentsData.splice(1,1);
console.log(studentsData);

Output:
[["Jack", 24]]

Iterating over Multidimensional Array
You can iterate over a multidimensional array using the Array's forEach() method to iterate over the multidimensional array.
let studentsData = [['Jack', 24], ['Sara', 23]];

// iterating over the studentsData
studentsData.forEach((student) => {
     student.forEach((data) => {
           console.log(data);
      });
});
