Conditional Statement

CONDITIONALS
Conditional statements control behavior in JavaScript and determine whether or not pieces of code can run.
There are multiple different types of conditionals in JavaScript including:

“If” statements:
- where if a condition is true it is used to specify execution for a block of code.
“Else” statements:
- where if the same condition is false it specifies the execution for a block of code.
“Else if” statements:
- this specifies a new test if the first condition is false.

Now that you have the basic JavaScript conditional statement definitions, let’s show you examples of each.


Explain each: The if statement checks if the condition num > 50 is true. If it is true, the code inside the if block will be executed.

In this example, since num is not greater than 50, the code inside the if block is skipped.

The else if statement is evaluated only if the preceding if condition is false. It allows you to specify an alternative condition to check.
If the else if condition is true, the code inside the corresponding block will be executed.

In this example, the condition num < 50 is true, so the code inside the else if block is executed, printing "The number is less than 50" to the console.

The else statement is optional and provides a default block of code to be executed when all preceding conditions are false.
If none of the previous conditions are true, the code inside the else block will be executed.

In this example, since num is neither greater than nor less than 50, the code inside the else block is executed,
printing "The number is equal to 50" to the console.

It's important to note that in this example, only one of the three blocks (if, else if, or else) will be executed based on the condition that evaluates to true. Once one condition is satisfied, the subsequent conditions are not checked.
