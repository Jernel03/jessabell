What are JavaScript Loops?
JavaScript loops are used to repeatedly execute a block of code until a specified condition is met.
They allow you to automate repetitive tasks and iterate over collections of data.
JavaScript provides several types of loops:

      - For Loops
      - While Loops
      - Do While Loops


For loop:
     The for loop is the most commonly used loop in JavaScript.
     It allows you to execute a block of code a specific number of times, usually based on a counter variable.

while loop:
     The while loop executes a block of code as long as a specified condition is true.
     It is useful when you don't know the exact number of iterations in advance.


     In this example, the loop will continue executing as long as i is less than 5.
     The i value is incremented inside the loop to avoid an infinite loop.

do-while
     The do-while loop is similar to the while loop, but it guarantees that
     the code block is executed at least once, even if the condition is initially false


     The code block is executed first, and then the condition is checked. If the condition is true, the loop continues.
     Otherwise, it terminates.
